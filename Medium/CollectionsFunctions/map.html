    Map

    Sintaxe¶

    Como vimos podemos aplicar transformações em coleções, filtra-las, gera-las, existe mais uma maneira de aplicar estes conceitos as listas e arrays em Python, podemos utilizar a função map(),mas antes devemos nos atentar aos parâmetros da função map, o primeiro deles é uma função que será aplicada a cada item da lista o segundo naturalmente deve ser a lista que vamos percorrer, existe mais um ponto que devemos ter em mente a função map retorna um valor do tipo map é uma coleção de hashs, ou seja, pares de chaves codificadas e o valores da transformação aplicada aos elemento da lista, portanto devemos transformar essa coleção de hashs em uma lista para apresentar o resultado,vamos ver um exemplo simples para começar.

    • Obs: Serão utilizadas funções e expressões lambda nos exemplos

    Exercícios¶

    1-) Multiplicar os itens de um array por um valor n arbitrário:

    array = [1,2,3,4,5]
n = int(input("Digite um numero para multiplicar os elementos do vetor: "))
resultado = list(map(lambda x: x*n, array))

print(*resultado, sep=" ")

    NOTE QUE: Foi utilizada a transformação do objeto do tipo map para um do tipo list com list( map( function, iterator ) ) .

    2-) Eleve ao quadrado os elementos ímpares de um array e multiplique por 5 os pares:

    array = [1,2,3,4,5]
resultado = list(map(lambda x: x * 5 if x % 2 == 0 else x ** 2,array))

print(*resultado, sep=" ")

    3-) Filtre o array de nomes por uma sequencia de caracteres fornecida pelo usuário, em seguida apresente os nomes:

    array = ["Marcos","Manuel","Monael","Jorge","Jade","Ana","Aline","João"]
filtro = input("Digite a sequncia de char`s para filtrar o array: ")

resultado = list(map(lambda nome: nome if nome.find(filtro) != -1 else None, array))
print(*resultado, sep=" ")

    NOTE QUE: A função map pode ser utilizada para filtrar listas, entretanto perceba que ela atribui o valor None aos objetos que não satisfazem a condição, isso gera uma lista com lixo dependendo do seu objetivo final, no caso o exercício pediu para filtrar e apresentar estes nomes, teríamos que realizar um segundo filtro no resultado para retirar os valores None de nossa lista.

    Porem existem aplicações para essa filtragem, veremos algumas delas abaixo:

    4-) Veja quantos elementos no array não satisfazem o filtro:

    Obs: Existem diferentes maneiras de resolver este problema, vamos utilizar map* neste primeiro exemplo: 

    array = ["Marcos","Manuel","Monael","Jorge","Jade","Ana","Aline","João"]
filtro = input("Digite a sequncia de char`s para filtrar o array: ")

array = list(map(lambda nome: nome if nome.find(filtro) != -1 else None, array))

print(f'{array.count(None)} elementos não satisfazem o filtro.')

    Vendo os exercicios acima chamo sua atenção para que não utilize map para filtar coleções, sua utilização deve ser para aplicar transformações nos elementos da coleção.
