    Dicionários

    Então vamos lá pessoal vamos entender a estrutura de dados chamada dicionário, são estruturas que funcionam com pares de chave-valor, ou seja, podemos nomear nossos registros nessa estrutura e utiliza-las para resolver problemas de maneira mais simples e com maior legibilidade, vamos ver como essa estrutura funciona.

    Sintaxe¶

    Usuario = {'Nome':'Jorge','Idade':23}

    NOTE QUE: Temos a declaração de uma variável do tipo dicionário e percebam que utilizamos o sinal de ' { } ' , além disso cada par de chave-valor é separado pelo caractere ' : '.

    Agora vamos entender como acessar , alterar e inserir os pares de chave-valor nessa estrutura, é importante dizer que devemos conhecer nossas entidades antes de pensar em utilizar essa estrutura.

    Usuario = {}

Usuario["Nome"] = "Guilherme"
Usuario["Nome"] = "Jorge"
Usuario["NomE"] = "André"

    Acima conseguimos inserir um novo par chave-valor para o usuário, ou seja, dizemos que para a chave "Nome" temos o valor "Guilherme". É importante dizer que podemos registrar mais pares caso seja necessário, perceba que logo em seguida na mesma chave "Nome" alteramos seu valor para "Jorge", entretanto ao definirmos "NomE" estamos definindo uma nova chave e passando a ela o valor "André". De maneira geral a sintaxe é essa, vamos ver como acessar, atualizar e remover itens de um dicionário.

    Métodos¶

    Update¶

    Atualização dos valores para as chave já inseridas no dicionário.

    dict = {"1":"Carlos","2":"Julio"}
print(f'Dicionario Original: {dict}')

dict.update({"1":"Marcos"})
dict.update({"2":"André"})
print(f'Dicionario Atualizado: {dict}')

    Del¶

    Apagando um par de chave-valor especifico, ou o próprio dicionário.

    DocCarro = {"Nome":"Carlos","Carro":"Celta","Placa":"ABC123"}
print(f'Documento do Carro Original: {dict}')

del DocCarro["Placa"]
print(f'Documento do Carro sem Placa : {dict}')

del DocCarro

    Pop - PopItem¶

    Aqui vamos ver duas funções que apagam a ultima aparição do registro que passamos como parâmetro e também os retorna.

    pessoa = {"Nome":"Paula","Sobrenome":"Oliveira","Idade":25}
idade = pessoa.pop("Idade")
print(f'Idade da pessoa: {idade}')

print(f'Pessoa sem sua idade: {pessoa.popitem()}')
print(f'Dicionario vazio: {pessoa}')

    Percebam que o dicionário no ultimo print() estará vazio pois utilizamos o método popitem(), e será apresentada sem sua idade pois utilizamos o método pessoa.pop("idade") .

    Clear¶

    Limpeza da estrutura apagando todos os pares chave-valor, sua diferença em relação ao del dict é que ele não destrói a a estrutura, apenas deixa ele vazio.

    dict = {"1":"Carlos","2":"Julio"}
print(f'Dicionario Original: {dict}')

dict.clear()
print(f'Dicionario Limpo: {dict}')

    Percorrendo Dicionário¶

    É importante dizer que podemos percorrer todos os subitens relativos a estrutura dicionário, itens, chaves e valores, vamos ver como isso funciona:

    aluno = {"Nome":"Kevin","Idade":25,"RA":"XXXXXXX-XX"}

for key in aluno.keys():
    print(f"Chave : {key}")

for value in aluno.values():
    print(f'Valor : {value}')

for key,value in aluno.items():
    print(key,value)

    Aconselho o uso destas funções para melhorar a semântica do seu código, mas vou mostrar outros dois jeitos de percorrermos as chaves e valores.

    carro = {"Nome":"Fiesta","Marca":"Ford","Ano":2019}

for key in carro:
    'Printa as chaves do dicionario'
    print(key)

for key in carro:
    'Printa os valores do dicionario'
    print(carro[key])

    Compressão de Dicionário¶

    Assim como as lista em que temos list comprehension com os dicionários temos dict comprehension, vamos dar uma olhada em como isso funciona com alguns exemplos.

    1-) Crie um dicionário que relacione os nomes com seus RAs formando um dicionário de alunos:

    nomes = ["Jorge","Marcos","Thales","Gabriel"]
rAs = ["xxxx","yyyy","hhhh","llll"]

alunos = { key : value for key,value in zip(rAs,nomes) }
print(alunos)

    2-) Crie um dicionário que converta um dicionário de temperaturas em Kelvin para um com as temperaturas em graus Celsius:

    tempKelvin = {'t1':273,'t2':0,'t3':1080}

tempCelsius = {key : value - 273 for key, value in tempKelvin.items()}
print(tempCelsius)

    3-) Crie um dicionário com os números Fibonacci indexados pela sua posição na sequencia:

    • Por uma questão de organização usaremos a fórmula fechada para a sequencia de Fibonacci em uma expressão lambda.
    limit = int(input("Digite o limite para o calculo da sequencia: "))

fibonacci = lambda n: int((1/(5**0.5)) * (((1 + 5**0.5)/2)**n - ((1 - 5**0.5)/2)**n))

dict_fibonacci = {key: fibonacci(key) for key in range(limit)}

print(dict_fibonacci)

    Exercício¶

    Vamos lá pessoa, um dos propósitos deste material é mostrar como melhorar nosso códigos e aprender a utilizar bem estruturas de dados, por isso vamos utilizar o mesmo exercício das que utilizamos na seção das matrizes.

    Imagine que você carrega em memória uma tabela de um banco de dados relativa as vendas e você precisa extrair alguns dados dessa tabela.

    Por simplicidade uma venda tem as seguintes características:

    • Nome do Cliente;
    • Nome do Vendedor;
    • Nome do produto;
    • Quantidade do produto;
    • Valor do produto;

    Queremos saber qual o cliente que mais compra, qual o produto que mais vende e qual o vendedor com a maior média no valor da venda.

    Vamos entrar com um conceito novo aqui, percebam que nos temos uma lista de vendas, vamos tratar as vendas como uma entidade agora, relacionando atributos a valores por meio de chave-valor.

    vendas = []
vendas.append({"Cliente":"Valter","Vendedor":"Alex","Prodt":"Papel-A4","Qtd":576,"Val":25.90})

vendas.append({"Cliente":"Valter","Vendedor":"Anderson","Prodt":"Borracha","Qtd":120,"Val":2.95})

vendas.append({"Cliente":"Thales","Vendedor":"Alex","Prodt":"Papel-A4","Qtd":180,"Val":25.90})

vendas.append({"Cliente":"Thales","Vendedor":"Alex","Prodt":"Papel-A4","Qtd":590,"Val":2.95})

    Acima temos a base de dados construída com uma lista de dicionários que tratam de modelar nossa entidade venda.

    Percebam que não precisamos de um cabeçalho e semanticamente a solução já esta melhor que a feita com matrizes.

    'Cliente que mais compra'
clientes = set([venda["Cliente"]  for venda in vendas])
info = []
for cliente in clientes:
    totalVendas = 0
    for venda in vendas:
        if(venda["Cliente"] == cliente):
            totalVendas += venda["Qtd"] * venda["Val"]
    info.append({cliente:totalVendas})

    Bom gostaríamos que você comparasse o código feito com dicionários com o mesmo código feito utilizando a matriz como uma tabela de vendas, as mudanças são bem pontuais, entretanto lendo o código não temos mais aqueles problemas com "números mágicos" relativos a coluna da tabela, agora com dicionários temos o poder de nomear as colunas e isso facilita nossa interpretação ao ler o algoritmo, como eu disse antes no tópico de matrizes veremos que existe outra forma de resolver esse problema com POO veremos isso somente nos tópicos avançados. Este exercício por se tratar de um exemplo que pode ser útil no cotidiano será utilizado em outros tópicos como por exemplo o intermediário onde iremos utilizar funções mais especificas para coleções, fazendo você capaz de resolver problemas do cotidiano de uma maneira mais sucinta, por hora vamos só aprender a utilizar as estruturas de dados.
