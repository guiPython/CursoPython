    Reduce

    Introdução¶

    Por fim nas operações com listas que utilizamos funções temos reduce, pode parecer confuso quando utilizar cada operação, mas ao final vamos tentar deixar mais claro com exemplos e casos de uso, vamos logo para o reduce então.

    Para utilizamos esta operação devemos realizar uma importação em nosso código, uma importação é basicamente você falando para o interpretador de Python:

    "Olha amigão o pacote básico que você me oferece não me satisfaz mais, traz aquele outro pacote."

    Vamos demonstrar a sintaxe desse comando para importar a função reduce, porém é importante dizer que podemos importar qualquer outra biblioteca ou função que esteja em um pacote Python com este comando como Pandas, NumPy e etc.

    from functools import reduce

    Sintaxe¶

    Bom basicamente temos dois parâmetros para função reduce o primeiro refere-se ao a função que iremos aplicar aos elementos de uma coleção, como o próprio nome diz a função ira reduzir a coleção a um único elemento por meio de um predicate ou function, pedicates são expressões de retorno bool e se um predicate for utilizado a um reduce o ultimo valor true será retornado, caso aplicarmos uma function teríamos basicamente um somatório, produtório entre outros, o segundo parâmetro refere-se a própria coleção.

    Abaixo temos um exemplo de reduce aplicado com uma function.

    from functools import reduce

numeros = [1,2,3,4,5]

soma = reduce(lambda soma, numero: soma + numero, numeros)
print(f'Soma = {soma}')

    Vamos logo ao que interessa que é aprender a utilizar reduce, bom ele lembra bastante o funcionamento do map e filter, porém como vimos map aplica uma transformação em todos os elementos da coleção baseado em uma condição, operação aritmética ou aplicação de funções e o retorno é uma coleção dos objetos transformados, o filter retorna uma coleção baseada em uma condição só o elementos que ás satisfazem serão inseridos na coleção de retorno, agora o reduce continua com o padrão de aplicar uma função a uma coleção, porém seu retorno é sempre baseado no retorno da operação aplicada, ou seja, se o retorno da expressão aplicada for int o método reduce retorna um int, vamos ver isso na prática:

    SHOW ME THE CODE:

    1-) Itere sobre uma lista de nomes, pegue e apresente o maior deles:

    from functools import reduce

nomes =  ["Marcos Antonio","Jade Testando","Ana Maluca","Aline Crazy Crazy","João DeBoa"]
maiorNome = reduce(lambda maiorNome, nome: nome if len(nome) > len(maiorNome) else maiorNome, nomes)
print(f'Maior nome é {maiorNome}')

    NOTE QUE: Nossa expressão lambda do exemplo acima retorna uma string baseada em uma condição, portanto o reduce irá retornar o ultimo elemento que satisfez a condição.

    2-) Apresente o ultimo numero par divisível por 7 da sequencia:

    from functools import reduce

numeros = [x for x in range(100000)]

valor = reduce(lambda valor, numero: numero if ( numero % 2 == 0 and numero % 7 == 0 ) else valor, numeros)

print(f'O maior valor par divisivel por 7 é  : {valor}'')

    3-) Some todos os número ímpares do array com a soma iniciada no numero 6:

    from functools import reduce

numeros = [x for x in range(100000)]

valor = reduce(lambda soma, numero: soma + numero if numero % 2 != 0 else soma, numeros, 6)

print(f'A soma de todos os valores ímpares de 0 a 100000 mais o valor 6 é {valor}')

    NOTE QUE: Inserimos um terceiro parâmetro na função reduce, ele é chamado de inital, se fornecermos uma lista vazia ao parâmetro iterator de reduce este valor será retornado, ou seja, se estivermos somando ao percorrer a coleção na primeira iteração temos initial mais o primeiro valor.
